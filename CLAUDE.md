# Project Purpose

This project contains notes on Italian Art for my upcoming travel

# Structure

- `NOTES.md` - Reading notes mainly following the textbook "History Of Italian Renaissance Art Painting, Sculpture, Architecture Hartt, Frederick 7th Ed 2011"
- `README.md` - Navigation index with links organized by location, artist, period, and medium. With a special terms section.
- `artists/` - Individual artist pages
- `locations/` - Individual location pages
- `artworks/` - Individual artwork pages
- `img/` - Local images (floor plans, diagrams, etc.)
