# Trinity

- **Artist**: [Masaccio](../artists/Masaccio.md)
- **Location**: [Santa Maria Novella](../locations/SantaMariaNovella.md), Florence
- **Medium**: Fresco

## Description

A revolutionary work demonstrating Brunelleschi's principles of linear perspective. Created with Masolino.
