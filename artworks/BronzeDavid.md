# Bronze David

- **Artist**: [Donatello](../artists/Donatello.md)
- **Location**: [Bargello](../locations/Bargello.md), Florence
- **Medium**: Bronze

## Description

First large-scale nude sculpture since antiquity. The pose is derived from contrapposto, with the left knee coming forward against the cloth.
