# Hercules and Antaeus (Bronze)

- **Artist**: [Antonio del Pollaiuolo](../artists/AntonioDelPollaiuolo.md)
- **Location**: [Bargello](../locations/Bargello.md), Florence
- **Medium**: Bronze

## Description

Bronze sculpture depicting Hercules wrestling with the giant Antaeus. Shows Pollaiuolo's mastery of depicting dynamic action in three dimensions.
