# North Door of Florence Baptistery

- **Artist**: [Ghiberti](../artists/LorenzoGhiberti.md)
- **Location**: [Florence Baptistery](../locations/BaptisteryOfFlorence.md), Florence
- **Medium**: Bronze

## Description

The second set of bronze doors for the Florence Baptistery, created after Ghiberti won the famous competition against Brunelleschi.
