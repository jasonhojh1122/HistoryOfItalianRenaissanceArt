# Competition Panel (Ghiberti)

- **Artist**: [Ghiberti](../artists/LorenzoGhiberti.md)
- **Location**: [Bargello](../locations/Bargello.md), Florence
- **Medium**: Bronze relief

## Description

Ghiberti's winning competition panel for the second set of bronze doors for the Florence Baptistery. Only the Isaac figure was cast separately, making the whole piece lighter but stronger.
