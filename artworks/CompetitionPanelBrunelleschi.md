# Competition Panel (Brunelleschi)

- **Artist**: [Brunelleschi](../artists/FilippoBrunelleschi.md)
- **Location**: [Bargello](../locations/Bargello.md), Florence
- **Medium**: Bronze relief

## Description

Brunelleschi's competition panel for the second set of bronze doors for the Florence Baptistery. The servant in the left's pose is derived from the Spinario, a popular Roman sculpture. Each figure was cast individually.
